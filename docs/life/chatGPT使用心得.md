# chatGPT使用心得

## 1.多种文本即时渲染语言

- Markdown：一种轻量级的标记语言，使用简单的文本格式来描述文档的结构和样式，可用于撰写日志、文档、报告等。
- Mermaid：一种基于JavaScript的开源工具，使用简单的文本语法来描述图表并实时生成图像。
- AsciiMath：一种用 ASCII 字符来描述数学公式的语言，能够将公式即时转化为图像。
- PlantUML：一种用文本来描述 UML 图表的语言，能够将 UML 图表即时转化为图像。
- LaTeX：一种排版语言，能够将复杂的数学公式、物理公式等排版成美观的文本和图像。
- Graphviz：一种用文本来描述图表的语言，能够将各种类型的图表即时转化为图像。
