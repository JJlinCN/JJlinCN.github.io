## VScode可折叠区域自定义

在日常代码编辑中，我们经常会遇到想要折叠一个代码块这种情形，以屏蔽暂时不需要的代码块进行概览，可以使用关键词配对进行指定代码区域屏蔽。具体文件类型中的关键词写法如下图所示：

![关键词图示](/assets/软件教程/vscode/1.png)

::: tip 注意
在使用关键字时，请确保待自定义折叠区域内的层级关系必须是从大到小，不然不能配对关键字起作用。
:::
